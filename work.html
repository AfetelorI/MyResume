My first project

This license project represents a web application for managing the services of a cafe. The application allows the management of orders, menus and reservations for the smooth operation of the cafe. Main functionalities include: user registration and authentication, menu viewing, online ordering, reservation management and inventory monitoring.

The application is built on an architecture that clearly separates responsibilities between front-end and back-end. The back-end was developed using ASP.NET Core in the Visual Studio development environment. For the database, PostgreSQL was used, which facilitates the storage and manipulation of the application's essential data. On the front-end, HTML, CSS and JavaScript were used to create an easy-to-use interface.
